# 高考志愿填报助手 - 基于张雪峰方法论

## 🎯 项目简介

这是一个基于**张雪峰高考志愿填报方法论**的AI对话助手，采用H5单页应用形式，可在微信中直接打开和分享。

### 核心功能

1. **智能对话系统**：像聊天一样逐步收集用户信息（省份、分数、位次、选科等）
2. **冲稳保策略推荐**：基于位次法自动生成冲、稳、保三档院校+专业组合
3. **专业就业分析**：内置专业数据库，提供就业率、薪资、前景等详细分析
4. **避坑指南**：识别天坑专业，提醒常见填报误区
5. **城市选择建议**：基于城市分级理论给出个性化建议
6. **微信友好**：支持微信内打开、转发分享

## 🚀 快速开始

### 方式一：直接在浏览器打开（最简单）

1. 双击 `index.html` 文件，即可在浏览器中使用
2. 或右键 → 选择浏览器打开

### 方式二：本地HTTP服务器（推荐）

由于微信分享需要HTTPS或合法域名，建议使用本地服务器进行开发测试：

#### 使用 Python（推荐）

```bash
# 进入项目目录
cd gaokao-volunteer

# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

然后访问：`http://localhost:8080`

#### 使用 Node.js

```bash
# 安装http-server（全局安装一次即可）
npm install -g http-server

# 启动服务
http-server -p 8080 -c-1
```

#### 使用 VS Code

安装 "Live Server" 插件后，右键 `index.html` → "Open with Live Server"

## 📱 微信使用方法

### 方法1：直接发送HTML文件（适合个人使用）

1. 将整个 `gaokao-volunteer` 文件夹压缩成zip
2. 通过微信发送给侄女
3. 侄女解压后用手机浏览器打开 `index.html` 即可使用

**优点**：简单直接，无需服务器  
**缺点**：每次需要重新发送文件

### 方法2：部署到公网服务器（推荐长期使用）

#### 选项A：GitHub Pages（免费）

1. 在 GitHub 创建新仓库（例如：`gaokao-volunteer`）
2. 上传所有文件到仓库
3. 进入 Settings → Pages → 选择 main 分支 → Save
4. 几分钟后获得访问地址：`https://你的用户名.github.io/gaokao-volunteer/`
5. 将此链接通过微信发送给侄女

#### 选项B：Vercel / Netlify（免费）

1. 访问 [vercel.com](https://vercel.com) 或 [netlify.com]
2. 导入GitHub仓库或拖拽上传文件夹
3. 自动获得公网可访问的URL
4. 将链接分享到微信

#### 选项C：云服务器（阿里云/腾讯云）

如果有自己的服务器：
1. 上传文件到Web目录（如 `/var/www/html/gaokao-volunteer/`）
2. 配置Nginx/Apache指向该目录
3. 申请域名并配置HTTPS（微信要求）
4. 将域名链接分享到微信

### 方法3：使用临时在线工具（快速演示）

1. 使用 [ngrok](https://ngrok.com) 或类似工具暴露本地端口
2. 获得临时公网URL
3. 将URL分享到微信（有效期有限）

## 🔧 技术架构

1. **zhang-xuefeng-prompt.js**
   - 张雪峰方法论的完整知识库

   - 多轮对话状态管理

   - 微信分享配置

   - 移动端优先的响应式设计
   - 渐变主题色（紫蓝渐变）
   - 流畅动画效果
   - 微信环境适配

- ✅ **纯前端实现**：无需后端服务器，可直接运行
- ✅ **零依赖**：不依赖任何第三方库（除微信SDK可选）
- ✅ **离线可用**：加载一次后可离线使用
- ✅ **隐私安全**：数据不离开用户设备
- ✅ **轻量级**：总大小 < 100KB

## 🎯 张雪峰方法论核心要点

### 1. 位次法（最重要）
> "你考了600分这句话没有意义，重要的是你排多少名"

- 用今年位次对比目标院校往年录取位次
- 关注近3年数据，看波动范围
- 新高考关注专业组位次

### 2. 冲稳保策略
- 冲：位次高2000-5000名的院校
- 稳：位次匹配（±2000名）的院校
- 保：位次高5000名以上的院校

### 3. 城市 > 学校 > 专业
> "能去一线尽量去一线，即使学校稍微差一点"

- 一线城市资源多、机会多、眼界开阔
- 但如果分数不够，宁去二线强势专业不去一线垃圾专业

### 4. 就业导向选专业
- 先看能不能找到工作，再看喜不喜欢
- 选朝阳产业，避开夕阳产业
- 利用家庭资源和人脉优势

### 5. 避开天坑
- 生化环材：本科几乎无法就业
- 管理类：门槛低替代性强
- 除非真爱科研且家庭条件好，否则慎重
